# 🧱 STEP 3 — Seed the Admin BTC Address (One-Time)

Create a **script**, not UI.

### `prisma/seed.ts`

```ts
import { PrismaClient } from "@prisma/client"

const prisma = new PrismaClient()

async function main() {
  await prisma.adminWallet.create({
    data: {
      asset: "BTC",
      address: "bc1qxxxxxxxxxxxxxxxxxxxx",
      network: "mainnet",
    },
  })
}

main()
  .catch(console.error)
  .finally(() => prisma.$disconnect())
```

Run:

```bash
npx prisma db seed
```

✔ This is your **single source of truth**

# 🧱 STEP 4 — Expose Deposit Address to Users (Read-Only API)

### `/app/api/wallets/btc/route.ts`

```ts
import { prisma } from "@/lib/prisma"
import { NextResponse } from "next/server"

export async function GET() {
  const wallet = await prisma.adminWallet.findFirst({
    where: { asset: "BTC" },
  })

  if (!wallet) {
    return NextResponse.json({ error: "BTC wallet not set" }, { status: 500 })
  }

  return NextResponse.json({
    address: wallet.address,
    network: wallet.network,
  })
}
```

Frontend shows:

* Address
* QR code
* Warnings
